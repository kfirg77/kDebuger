kDebuger
========

Show information about the current page

Usage: 

  You can use this plugin to manually log data. 

  $tmp=array('Arr A'=>1);
  kDebuger::log($tmp);

  $tmp=array('Arr B'=>2);
  kDebuger::log($tmp);    

  kDebuger::log('A');
  kDebuger::log('B');
  kDebuger::log('C');

  kDebuger plugin print array in slider ander
